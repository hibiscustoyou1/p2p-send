# 📦 全栈应用快速启动模板 (Full-Stack App Template)

## 📌 产品定位
本模板旨在提供一个**高生产力、强类型安全与高协作效率**的全栈开发底座。让任何基于 Vue 3 + Node.js 的新项目都能够迅速跨过“繁杂的基础设施配置”，直接进入核心业务的开发阶段。

## 🎯 核心解决痛点
1. **类型断层**：前后端接口联调时，类型经常不匹配，或者需要手动复制接口及传参定义。
2. **构建缓慢**：大型项目中，传统的 Webpack 等构建工具速度太慢，需使用现代化的 Vite 与 tsup/esbuild 构建。
3. **架构混乱**：跨端代码互相污染（如前端意外引入 fs 模型，或后端引用了 DOM）。
4. **状态与路由碎片化**：未整合状态管理与路由设计的最佳实践。

## 👥 目标使用对象
- 需要快速起步全栈项目（C 端或 B 端管理系统）的开发者。
- 注重开发体验、代码质量、强类型推断与极速热更新的敏捷团队。

## ✨ 核心特性
- **End-to-End Type Safety**：利用 PNPM Workspaces 和共享包，服务端的 Prisma/API 接口类型直接无缝流转到前端，一处修改全局编译报错预警。
- **Modern Stack**：Vue 3 Composition API, Tailwind CSS, Express v5, Prisma ORM。
- **Security Vault**：内置 `.env.enc` 加解密脚本，敏感配置不再明文提交。
